### 问题3：边界条件处理不当

- 

  **表现**：数组越界错误

- 

  **例子**：使用`{8,1,2,2,3}`测试题目3，数字8超出范围导致下标越界

-

  **原因**：对循环变量的作用域理解不深

### 问题5：方法调用错误

- 

  **表现**：静态与非静态方法调用混淆

  **例子**：在main方法中直接调用非静态方法

  **原因**：对Java的OOP概念理解不够
